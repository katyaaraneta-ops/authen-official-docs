# Authen.ai Official Documents

This repository holds the official, certified documents used by the Authen.ai Issuance Registry of Authen-Testnet. Each file in the `docs/` folder corresponds to a document that has been cryptographically certified on-chain.

## Structure

- `docs/` — Certified documents (PDFs, images, etc.) that can be verified via Authen.ai

## Usage

Links to these documents appear in verification records and crisis briefs as the **Official source**. Recipients can open the link to view the original certified document.

## Adding new documents

1. Add the file to the `docs/` folder.
2. Certify the same file in the Authen.ai Scribe.
3. The filename in this repo must match the filename used at certification.
